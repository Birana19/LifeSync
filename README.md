LifeSync - Habit Tracker App
LifeSync is an Android-based habit tracker application built using Android Studio and Kotlin. This app helps users build and maintain positive daily habits by allowing them to create habits, mark their daily progress, and stay motivated through consistency tracking. The simple and intuitive UI ensures smooth interaction and effortless habit management. 

Features: 
- Add and Manage Habits: Create new habits with ease and view all habits in a clean and organized list.
- Daily Habit Completion: Mark habits as completed each day and track daily achievements at a glance.
- Consistency Tracking: View progress and streaks over time, stay motivated by visualizing your habit completion history.
- Local Data Storage: Uses SharedPreferences to store and preserve user data, all habits and progress remain saved even after closing the app.
- Cleand and User-Friendly UI: Minimal and intuitive interface, smooth and simple navigation.

Tech Stack: 
- Language: Kotlin
- IDE: Android Studio
- Storage: SharedPreferences
- UI: XML Layouts
